To set up the server:


1. In the command line, enter: make

2. To run the program, enter in the command line: ./fileserver [port number]



To set up the client:


1. To give permissions to the program, command line: chmod +x fileclient.py

2. In the command line, enter in:
python3 fileclient.py [host] [port] [command] [file] [data port]


Note: Each must be run on different servers.
